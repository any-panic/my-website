コーディング・デザイン：制作詳細

カテゴリー

企業コーポレートサイト（警備・サービス業）

作成期間

1週間（デザイン：2日 / コーディング：4日）

使用したツール

Figma / Visual Studio Code / HTML / CSS / JavaScript

制作概要

信頼感と力強さが求められる警備関連会社のWebサイト制作を担当。既存のイメージを一新し、若年層の採用も意識したクリーンでモダンなレイアウトを採用しました。

サービス設定

地域密着型の警備サービス。施設警備からイベント警備、さらには「学生応援」などの地域貢献活動まで幅広く展開している企業の強みを可視化しています。

ターゲット

信頼できる警備会社を探している法人担当者。地域貢献に関心があり、活気のある職場を求めている求職者（学生・フリーター）。

目的

企業の信頼性と「誠実さ」を視覚的に伝える。学生応援プロジェクトなどの取り組みをアピールし、採用ブランディングを強化する。協力企業ロゴを掲載することで、地域ネットワークの広さを証明する。

コンセプト

「安心を、もっと身近に。誠実を、カタチに。」堅苦しい警備業のイメージを、鮮やかなブルーとホワイトのコントラストで払拭し、安心感とプロフェッショナリズムが共存するデザインを目指しました。

サイト構成

TOP：躍動感のあるメインビジュアルで事業のスケール感を提示。ABOUT US：企業のミッションと姿勢を簡潔に紹介。WORK：実際の現場写真を用い、具体的な業務内容を可視化。協力企業：パートナー企業を一覧化し、社会的信頼度を向上。

デザイン

信頼を象徴する「ロイヤルブルー」をメインカラーに設定。直線的なレイアウトで規律正しさを表現しつつ、写真には角丸や円形の要素を一部取り入れることで、地域に寄り添う「柔らかさ」も演出しています。
