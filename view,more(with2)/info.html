コーディング・デザイン：制作詳細

カテゴリー

企業コーポレートサイト（警備業・リブランディング案）

作成期間

2週間（デザイン：2日 / コーディング：6日）

使用したツール

Figma / Visual Studio Code / HTML / CSS / JavaScript

制作概要

「変わらない日常のために、変わり続ける警備を。」というキャッチコピーを軸に、従来の警備業界の堅苦しいイメージを払拭する、透明感のあるWebデザインを制作。
情報の優先順位を整理し、ユーザーが直感的に企業の姿勢を理解できる構成にしました。

サービス設定

・先進的な教育体制と、地域貢献（学生応援）を強みとする警備会社。
・DX化や新しい働き方を取り入れ、業界のイメージアップを図っている先進的な企業像を設定。

ターゲット

・従来の警備会社に「古さ」や「硬さ」を感じている潜在顧客。
・清潔感や教育体制を重視する採用候補者。

目的

  • ヒーロービジュアルでのキャッチコピーと、女性警備員の起用により、親しみやすさと安心感を提示する。
  • 「Message」セクションに黒背景を用いることで、トップからの流れにコントラストをつけ、企業の思いを強く印象付ける。

コンセプト

「透明性と進化」 余白を広く取ったミニマルなレイアウトにより、企業の「誠実さ」と「クリーンな社風」を視覚的に表現。

サイト構成

Hero Section：規律性の高いキャッチコピーと印象的な画像で世界観を提示。
News：最新情報をシンプルに配置し、更新性の高さをアピール。
Service icons：アイコン化し、事業の3本柱を明確化。
Message：企業の想いをじっくり読ませる設計。
Partner Logos：協力企業のロゴを配列。

デザイン

フォントは可読性の高いゴシック体を採用。装飾を最小限に抑えることで、メッセージと写真を際立たせています。
情報のグループ化を徹底し、スクロールするだけで企業の概要が自然と頭に入る「ストーリー性」を意識したUIデザインです。
