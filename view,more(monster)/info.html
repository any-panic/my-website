コーディング・デザイン：制作詳細

カテゴリー

インタラクティブ・カードデザイン（UIパーツ）

作成期間

1日（デザイン：3時間 / コーディング：5時間）

使用したツール

HTML / CSS / JavaScript

制作概要

視線が動くギミックを取り入れた、遊び心のあるデジタルプロフィールカード。単なる情報の提示にとどまらず、ユーザーの「つい触りたくなる」心理を突いたインタラクティブなUIを試作しました。

サービス設定

クリエイターのポートフォリオサイトや、SNSのプロフィールリンク集（Linktree等）での使用を想定したパーソナル・IDカード。

ターゲット

・デザインにこだわりを持つクリエイター。
・訪問者に強い第一印象を残したい個人ブロガーやエンジニア。

目的

  • ユーザーのマウスカーソルの動きに合わせて「目」が追従するJavaScriptの実装により、サイト滞在時間とエンゲージメントを向上させる。
  • パントーン・カラーサンプルのようなミニマルな構図を採用し、高いデザイン性と情報の整理を両立する。

コンセプト

「見つめる・繋がる（Eyecatching Identity）」 無機質なデジタルデータに「視線」という生きた要素を加えることで、画面越しでも体温が伝わるようなコミュニケーションを演出。

デザイン

パステルミントとアイボリーの配色で、クリーンかつモダンな印象に仕上げました。フォントには太めのゴシック体を採用し、タイポグラフィの配置をポスターデザインのように構成することで、狭い画面内でも強いインパクトを与えます。
